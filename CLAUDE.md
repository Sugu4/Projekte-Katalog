# Projekt-Katalog — Anweisungen für Claude

## Schreibstil
- Keine Gedankenstriche (—) im generierten Text verwenden
- Klare, direkte Sprache
- Deutsch als Hauptsprache
- Immer korrekte deutsche Umlaute verwenden: ä, ö, ü, Ä, Ö, Ü, ß (UTF-8)
- Niemals Ersatzschreibweisen wie ae, oe, ue, ss statt der echten Umlaute

## Projekt-Kategorien
- Webseiten
- Update für Webseiten
- Branding
- SaaS
- Bilder bearbeiten
- Videos bearbeiten
- Werbung für Unternehmen
- Google Konto Pflegen
- SEO
- Monatsberichte

## Design-Vorgaben
- Logo-Dateien liegen in: assets/
- Projekt-Bilder liegen in: assets/projekte/
- Ausgabe-Dateien (HTML, PDF) in: output/

## Workflow
1. Projekte hinzufügen: python add_project.py
2. Katalog exportieren: python generate_catalog.py
